jsTree Service FrameworkTSF 소스 다운로드

    • Apache Maven Resources Plugin 3.1.0은 소스 형식으로 배포됩니다. Apache Maven Resources Plugin을 직접 빌드하려면 소스 아카이브를 사용하십시오. 그렇지 않으면 중앙 저장소에서 준비된 2 진 아티팩트를 사용하기 만하면됩니다.
    • TSF는 소스 코드 형식으로 배포됩니다. TSF를 직접 빌드하려면 소스 아카이브를 사용하십시오.
    • 그렇지 않으면 중앙 저장소에서 준비된 2진 아티팩트를 사용 할 수 있습니다.
    • 또한, 클라우드 환경을 위한 도커 이미지를 Docker Hub 또는 빌드 이미지로 생성 할 수 있습니다.
